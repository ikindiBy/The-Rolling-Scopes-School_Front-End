How to play:

It is a counter game where the object is to find pairs.
Whene the game begins, all pictures are hidden.
To Play:

  1. Selected two cards to try to match the pictures.
  2. If you match the pictures you can go again.
  3. If they don't match it is the computer turn them.
  4. You have 3 minutes for finding all matches.
  5. The player that finds all pairs wins!
  6. Have Fun!
